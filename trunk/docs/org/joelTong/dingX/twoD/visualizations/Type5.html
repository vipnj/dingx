The class Type4 creates a typical line spectrum analyzer. However, the "line" is drawn using 512 particles and not the actual line drawing API found in Flash. There are 512 emitters, each emitter emitting a variable number of particles subject to changes in beat from the sound spectrum. The rate of update is subject to a combination of variables, namely: minParticles, maxParticles, fps, lifeTime, blurX and blurY.

public function Type5(w:int, h:int, minParticles:int = 2, maxParticles:int = 4, fps:int = 24, lifeTime:Number = 0.5, gravity:int = 5, timerUpdate:int = 50, amplitude:int = 50, performFFT:Boolean = false, blurX:int = 2, blurY:int = 2, radiusDot:int = 10, colorPattern:ColorPattern = null)

This constructor instantiates a Type4 object. It features a color mode that can be set by passing in an instance of the ColorPattern object. To use visualization, add it to an InitializerTwoD object.

Certain variables will dramatically alter performance. Please refer to comments for more details. Parameters

w:int — width of visualizer
 
h:int — height of visualizer
 
minParticles:int (default = 2) — Minimum number of particles emitted. Will affect update rate.
 
maxParticles:int (default = 4) — Maximum number of particles emitted. Will affect update rate.
 
fps:int (default = 24) — Optimal FPS to achieve. Will affect performance.
 
lifeTime:Number (default = 0.5) — Maximum lifetime of particle. Will affect performance.
 
gravity:int (default = 5) — Gravity to use on particle in the Y-axis direction.
 
timerUpdate:int (default = 50) — time to update sound spectrum data. Will affect performance.
 
amplitude:int (default = 50) — Amplitude of wave.
 
performFFT:Boolean (default = false) — To perform Fast Fourier Transform on sound spectrum.
 
blurX:int (default = 2) — blur in X axis direction. Will affect performance significantly.
 
blurY:int (default = 2) — blur in Y axis direction. Will affect performance significantly.
 
radiusDot:int (default = 10) — radius of dot to use.
 
colorPattern:ColorPattern (default = null) — color pattern to use. If not passed, it uses orange.
