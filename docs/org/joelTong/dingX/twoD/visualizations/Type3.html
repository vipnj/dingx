The class Type3 creates a linear bar visualization made up of particles. Particles are thrown up like a fountain according to magnitude of beat. Number of particles emitted depends also on the magnitude of sound output according to frequency.

public function Type3(w:int, h:int, noChannels:int = 10, barH:int = 20, noParticles:int = 400, lifetime:Number = 5, explosionThreshold:Number = 70, gravity:int = 100, explosionExponent:int = 20, fountainForce:int = -500, updateTime:int = 50, colorPattern:ColorPattern = null)

The Type3 constructor creates an instance of a linear bar sound spectrum analyzer in the form of particle fountains. It features a color mode that can be set by passing in an instance of the ColorPattern object. To use visualization, add it to an InitializerTwoD object.

Parameters
w:int — width of visualization.
 
h:int — height of visualization.
 
noChannels:int (default = 10) — number of channels analyzed in sound spectrum.
 
barH:int (default = 20) — height of particle jet acceleration area and particle generation area.
 
noParticles:int (default = 400) — maximum number of particles emitted per channel.
 
lifetime:Number (default = 5) — maximum lifetime of each particle.
 
explosionThreshold:Number (default = 70) — magnitude of threshold, out of 100, to emit particles. Useful for beat detection.
 
gravity:int (default = 100) — Gravity imposed on particles.
 
explosionExponent:int (default = 20) — Maximum jet acceleration used. Must be a negative number.
 
fountainForce:int (default = -500) — Maximum force of fountain.
 
updateTime:int (default = 50) — Time to refresh visualizer, in milliseconds. Shorter time means higher demand on CPU side.
 
colorPattern:ColorPattern (default = null) — The color scheme to use, passed as a ColorPattern object.
