public function Type2(w:int, h:int, noChannels:uint = 5, noParticles:int = 40, explosionPower:Number = 1.7, explosionThreshold:int = 70, particleWidth:int = 20, particleHeight:int = 20, particleBlurRadius:int = 4)

This constructor creates a visualization made up of rectangular particles. This was insipred by Type1 and is in celebration of new year's day 2010. It is based on the Flint Particle system developed by Richard Lord. To use visualization, add it to an InitializerTwoD object.

Parameters
w:int — width of visualization.
 
h:int — height of visualization.
 
noChannels:uint (default = 5) — number of channels of sound spectrum to analyze.
 
noParticles:int (default = 40) — number of particles used per channel. The more the merrier but needs more processing power.
 
explosionPower:Number (default = 1.7) — the power of the explosion coming from speaker.
 
explosionThreshold:int (default = 70) — Threshold of explosion. This is set for beat detection.
 
particleWidth:int (default = 20) — Width of rectangle. Note that it is a random value from value specified.
 
particleHeight:int (default = 20) — Height of particle.Note that it is a random value from value specified.
 
particleBlurRadius:int (default = 4) — Blue filer radius used on particle.
